## What is TypeORM?

TypeORM is an [ORM](https://en.wikipedia.org/wiki/Object-relational_mapping) that runs in Node.js, the browser, Cordova, Ionic, React Native, NativeScript, Expo, and Electron. It works with TypeScript and JavaScript (ES2021+) and supports both the **Active Record** and **Data Mapper** patterns.

## Key features

- **Active Record and Data Mapper** — choose the pattern that fits your architecture
- **TypeScript-first** — full type safety with decorator-based entity definitions
- **Powerful QueryBuilder** — build any SQL query with a fluent, chainable API
- **Automatic migrations** — generate and run schema migrations from your entity changes
- **Relations** — OneToOne, OneToMany, ManyToOne, ManyToMany with eager/lazy loading
- **Lifecycle hooks** — BeforeInsert, AfterLoad, and more for entity event handling
- **Query caching** — built-in caching with Redis and other providers
- **CLI** — scaffold projects, generate migrations, sync schemas from the command line

## Quick example

Define your entity with TypeScript decorators:

```typescript
import { Entity, PrimaryGeneratedColumn, Column } from "typeorm"

@Entity()
export class User {
    @PrimaryGeneratedColumn()
    id: number

    @Column()
    firstName: string

    @Column()
    lastName: string

    @Column({ default: true })
    isActive: boolean
}
```

Then query it with the Repository API:

```typescript
const userRepository = AppDataSource.getRepository(User)

// Save a user
const user = userRepository.create({ firstName: "Alice", lastName: "Smith" })
await userRepository.save(user)

// Find users
const users = await userRepository.find({ where: { isActive: true } })

// Update a user
await userRepository.update(1, { firstName: "Bob" })

// Remove a user
await userRepository.delete(1)
```

Or use the fluent QueryBuilder for complex queries:

```typescript
const users = await AppDataSource
    .getRepository(User)
    .createQueryBuilder("user")
    .where("user.isActive = :isActive", { isActive: true })
    .orderBy("user.lastName", "ASC")
    .getMany()
```

## Installation

<CodeGroup>
```bash npm
npm install typeorm reflect-metadata
```
```bash yarn
yarn add typeorm reflect-metadata
```
```bash pnpm
pnpm add typeorm reflect-metadata
```
</CodeGroup>

Then install the driver for your database — see [Getting Started](/getting-started) for driver-specific instructions.

<Tip>
  TypeORM requires `reflect-metadata` to be imported once in your application's entry point before any entities are loaded.
</Tip>
